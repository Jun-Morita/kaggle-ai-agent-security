# Knowledge Notes

コンペ中に参照する知識ベースです。

Codex には、まず `INDEX.md` を読ませる。関連する情報がある場合だけ、`notebooks.md`、`discussions.md`、`external_ideas.md`、`sdk_docs.md` を開く。

## 書き方

```markdown
## YYYY-MM-DD: title

- Source:
- Fetched at:
- Author:
- Type: notebook / discussion / documentation / article / paper
- Competition:

### Key Ideas
- 

### Useful for This Competition
- 

### Risks / Caveats
- 

### Experiment Candidates
- 
```

## 注意

- 事実、推測、実験候補を混ぜない。
- LB / replay / leakage / rule に関わる内容は特に慎重に書く。
- 使った知識は実験ノートにも出典を残す。
- 重要な知識を追加したら `INDEX.md` も更新する。
